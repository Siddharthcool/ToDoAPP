# To-Do App

## Description

The To-Do App is a task management application that allows users to create, update, and manage their tasks efficiently. 
It provides features to mark tasks as completed, delete tasks, and automatically suggest icons based on task names. 
Built with Flutter, this app ensures a smooth and responsive user experience.

## Features

- **Add Tasks**: Create new tasks with customizable names.
- **Mark as Completed**: Check off tasks as completed with a strikethrough effect.
- **Delete Tasks**: Remove tasks from your list.
- **Customizable Text**: Adjust font size and weight for task names.
